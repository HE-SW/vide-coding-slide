Claude Code 시스템 프롬프트

2026년 3월에 유출됐던 — 이미 박혀 있는 안전장치들

1
NEVER run force push to main/master
메인 브랜치 force push 원천 차단 — 팀 작업 기록을 통째로 덮어쓰는 사고 방지
2
NEVER use git commands with the -i flag
interactive 모드 사용 금지 — 사람 키 입력이 필요한 명령은 자동화 흐름을 끊는다
3
ALWAYS create NEW commits rather than amending
기존 커밋 수정 대신 항상 새 커밋 생성 — 히스토리 보존, 되돌리기 가능
여러분이 Claude Code를 실행하는 순간,
이미 하니스는 작동하고 있었습니다.
