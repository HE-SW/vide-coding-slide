② 충분한 맥락 (Context)

프롬프트가 "무엇을 시킬지"라면, 맥락은 "그 일을 하려면 알아야 할 것"입니다.

맥락은 AI에게 같이 던져주는 배경 자료·상황·이전 대화입니다.

· 관련 문서·메일 본문
· 받는 사람이 누구인지
· 우리 입장·제약 조건
· 지금까지 이뤄진 합의
맥락 없이 시키면 AI는 추측한다.
가장 흔한 케이스로 알아서 채워 넣음.
같은 프롬프트라도 배경 자료를 함께 주면
결과가 우리 상황에 딱 맞게 좁혀진다.
❌ 맥락 없이내일 A사 박 차장에게 보낼 팔로업 메일 써줘.
AI안녕하세요 박 차장님, 어제 자리 감사드리며 논의된 사항 정리해 공유드립니다. 1. … 2. … (무슨 회의였는지·합의가 뭔지 모름 → 빈칸 채워서 어색)
⭕ 맥락 제공[배경] 오늘 A사와 12월 출시 일정 미팅 [합의] 11월 말까지 API 테스트 완료 [미해결] 보안 검토 일정 조율 [관계] 박 차장과는 2년 협업 · 친근한 톤 OK 위 내용으로 박 차장에게 팔로업 메일 써줘.
AI박 차장님, 오늘 미팅 감사했습니다. 합의대로 11월 말까지 API 테스트는 저희가 책임지고 진행하겠습니다. 다만 보안 검토 일정은 다음 주 중 별도 조율이 필요할 것 같은데, 가능하신 시간 알려주시면 맞춰보겠습니다.
관찰똑같은 요청, 배경 자료가 붙으면 결과가 우리 일에 바로 쓸 수 있게 바뀐다.
